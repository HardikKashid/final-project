# 🧠 Burnout Wellness Dashboard

## 📌 Overview

Burnout Wellness Dashboard is a smart web-based system designed to monitor daily habits, detect early signs of burnout, and provide actionable insights to improve mental and physical wellbeing.

The system combines structured routines, real-time check-ins, and interactive tools to help users maintain balance and avoid burnout.

---

## 🚀 Features

### 🌅 Morning Routine

* Track sleep hours and quality
* Log mood and daily habits
* Activity checklist (meditation, exercise, hydration, etc.)
* Progress tracking

### 🌙 Evening Routine

* Wind-down and bedtime tracking
* Relaxation activity monitoring
* Mood reflection

### 📋 Daily Check-in

* Stress, energy, and productivity sliders
* Mood tracking
* Water intake tracker
* Daily reflection journaling

### 🧰 Wellness Tools

* Breathing exercise (4-7-8 technique)
* Pomodoro focus timer
* Mood journal
* Daily affirmations

### 📊 Dashboard Analytics

* Burnout risk assessment
* Real-time recommendations
* Weekly stress chart
* Habit tracking & streaks

---

## ⚙️ How It Works

1. User logs daily routines (morning & evening)
2. Inputs mood, stress, energy, and activities
3. System analyzes patterns
4. Burnout risk is calculated
5. Personalized recommendations are shown

---

## 🛠️ Tech Stack

* HTML
* CSS (advanced UI with animations & glassmorphism)
* JavaScript (logic + interactivity)
* Browser localStorage (data storage)

---

## ▶️ How to Run

1. Open `index.html` in any browser
2. Navigate between sections (Morning, Evening, Check-in, Dashboard)
3. Enter your daily data
4. View insights and burnout analysis

## 💡 Problem Solved

Most people experiencing burnout don’t actively track their health.

This system solves that by:

* Making tracking simple and engaging
* Providing instant insights
* Encouraging healthy routines
* Offering recovery tools in one place
